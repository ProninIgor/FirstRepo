## Simple Budget
### Вступление
###### Контролировать свои финансы - это просто.  
#
*Вдохновлён книгой "Самый богатый человек в Вавилоне(Д.С.Клейсон, 1926г)"*

Раньше в моей семье были проблемы с финансами. Постояннно возникали вопросы: 
 - *"Куда делись деньги?"*
 - *"Где взять деньги на покупку вещей?"*.

У вас такие же вопросы, то приложение **Simple Budget** поможет ответить на эти вопросы.

### Описание
На чём строиться успех: 
- Правила обращения с деньгами
- Прозрачность доходов и расходов. 

Инструменты: 
 - Список простых правил из книги 
- Вы вносите каждый месяц дохоы и расходы
- Ставите категории для каждой операции доходов и расходов.
- Планируете расходы для месяц вперёд
- Отслеживаете плановый баланс накоплений и баланс расходов на месяц
- Многопользовательский доступ для всех членов семьи.
- Плановый список покупок на день.
- Аудит
- Месячные отчёты
